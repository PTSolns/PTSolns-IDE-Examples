# PTSolns IDE Examples

This repository contains example sketches and projects that demonstrate how to use the PTSolns IDE with various microcontroller boards and peripherals.

Included are:
- `examples/` — curated sketches showing common use cases
- `extras/` — additional helpers or supplementary example files
- Scripts and tools to format or organize example code

These examples are designed to work with the PTSolns IDE and can help you get started building your own microcontroller projects quickly.

## Download

Download the latest PTSolns IDE release here: [_PTSolns IDE_](https://ptsolns.com/ide).
